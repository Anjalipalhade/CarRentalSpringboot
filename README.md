# Car Rental System Project

## Description
This is a Java + Spring Boot + Maven project for managing car rentals. It includes:

- Backend: Spring Boot, Java, JDBC/MongoDB
- Frontend: HTML, CSS, JS
- Database: MongoDB

## Features
- Add, update, delete cars
- Book cars
- User login and registration
- View rental history
